# WhatsCookin
**An Intelligent Culinary Engine that transforms raw ingredients into structured, cookable recipes.**

Link to demo: https://drive.google.com/file/d/1YnZ4Q8qp4BuhHuttINg0jXwt6uXUCZfA/view?usp=drive_link

* Frontend: Flutter (State management, fluid UI transitions).
* Backend: Node.js (Asynchronous request handling).
* AI Layer: Llama (Few-shot prompting for culinary logic).
* Integrated Google Search API to verify ingredient pairings and improve relevance.
* Deployment: Docker-containerized for scalability.
